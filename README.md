
# Supermarket
Supermarket App MVVM

## Description
I made an app in C# using Entity Framework and WPF (Windows Presentation Foundation).

## Features
- Two modules: 
  - Admin
  - Cashier

### Admin Module
- Add, delete, update, and view various objects.
- Perform various operations.
